## Project structure

#### Backend
* Platform: Java 11
* Framework: Spring
* Package management tool: Maven

#### Frontend
* Platform: HTML / CSS / Javascript
* Framework: 
	* ReactJs - Redux
	* Bootstrap
* Package managament tool: NPM

#### Database
* MYSQL

## Directory structure
* **backend:** contains backend source code
* **frontend:** contains frontend source code
* **docs:** contains document, diagrams, reports ...
